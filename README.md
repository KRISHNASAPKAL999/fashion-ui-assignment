## 📋 Description

Fashion UI Assignment is a modern, responsive e-commerce landing page designed to showcase fashion products with a sleek and contemporary user interface. Built with pure HTML5 and CSS3, this project demonstrates advanced frontend design principles including responsive layouts, modern positioning techniques, and beautiful UI components.

The project showcases a complete fashion store interface with an interactive navigation bar, stunning hero section, and elegantly designed product showcase cards.

## 🚀 How to Run

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional installations required!

### Steps

1. **Clone or Download** the project files:
   ```bash
   git clone https://github.com/yourusername/fashion-ui-assignment.git
   cd fashion-ui-assignment
   ```

2. **Open the HTML file**:
   - Double-click `Assignment.html` to open in your default browser, OR
   - Right-click `Assignment.html` → Open with → Choose your preferred browser

3. **View in VS Code Live Server** (Optional):
   ```bash
   # Install Live Server extension in VS Code
   # Right-click on Assignment.html
   # Select "Open with Live Server"
   ```

## 📝 Code Example

### Basic HTML Structure
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Fashion UI Assignment</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <!-- Navigation and content here -->
</body>
</html>
```

### CSS Positioning Example
```css
/* Relative Positioning */
.product-card {
    position: relative;
    width: 300px;
    height: 400px;
}

/* Absolute Positioning for floating elements */
.floating-element {
    position: absolute;
    top: 20px;
    right: 20px;
    animation: float 3s ease-in-out infinite;
}

/* Flexbox Layout */
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    padding: 1rem 2rem;
}
```
